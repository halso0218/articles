# Youtube広告スキップをショートカット化！

## tl;dr
YouTubeはショートカットキーで操作できて便利ですが、広告のスキップだけはマウス操作が必要です。
そこで登場するのが AutoHotKey。ショートカットキーをカスタマイズできる有名なツールで、任意のキー入力に対して指定座標をクリックする操作を割り当てられます。これを使って広告スキップを自動化します。

## 手順
1. AutoHotKeyのインストール
2. 「スキップ▶|」ボタンの座標を取得
3. 広告スキップ用の .ahk ファイルを作成
4. スクリプトを常駐させる

## 1. AutoHotKeyのインストール
[AutoHotKeyのサイト](https://www.autohotkey.com/)からダウンロードしてください。  
ver1.1とver2.0がありますが、設定ファイル(スクリプト)に互換性がないので、今後のことを考えて**ver2.0をインストールする**ようにしましょう。

## 2. "スキップ▶|"ボタンの座標の取得
FullHDで全画面表示の場合、座標はおおよそ (1800, 945) に表示されます。  
座標はAutoHotKeyで簡単に取得できます。以下のスクリプトを getpos.ahk などの名前で保存し、ダブルクリックで実行してください。

```ahk
F1:: {
    MouseGetPos &xpos, &ypos
    MsgBox "X座標: " xpos "  Y座標: " ypos
}
```
使い方:
「スキップ▶|」ボタンにカーソルを合わせて F1キー を押すと座標が表示されます。メモしておきましょう。

## 3. 広告スキップ.ahkファイルの作成
以下のスクリプトを 広告スキップ.ahk などの名前で保存します。
この例では F1キー にスキップ操作を割り当てています。

```ahk
F1:: {
    Click(1800, 945) ; ご自身の環境にあわせて座標を修正してください
}
```
既にAutoHotKeyを使っている場合は、普段の .ahk ファイルにこのコードを追加してもOKです。

## 4. スクリプトを常駐させる
作成した 広告スキップ.ahk をダブルクリックして常駐させれば準備完了です。
