# 一般口座で米国株式を買った顛末

「まあ、売るとき考えればいいっしょ」と考えていた私は、こんなに苦労することになるとは思いもしなかった。以下が事の顛末だ。

# 円資産だけじゃヤバイ！
今から4年前の2022年のはじめ頃、いつまでたってもコロナの幻影に怯える民をみて「もうちょっと日本ヤバいんじゃない」という考えに支配されていた私は、資産逸失のリスクを減らすためにドルを買うことにした。ちょうど急激な円安が始まった頃だ。
いつものように、思ってから動き出すまでに一月ほど時間がかかったが4月6日から毎週5,000円ずつ積み立てることにした。

# ドル預金だけじゃヤバイ！
このころの私は、「現金だけだとインフレに対応できないからヤバイんじゃないか」という考えにも支配されており、どんどんたまっていくドルをそのままにしておくことに対して不安になっていた。
個別株買うほどの額でもないのでETFでも買っておけばいいや面倒だし、ということで、高配当株っていうのがあるらしいというのを知った。いわゆるHDV,SPYD,VYMのことだ。

# そんな銘柄ないんだけど！
「さて、ある程度ドルがたまってきたからHDVでも買ってみますか」ということでSBI証券でHDVと検索してみる。ところが検索結果に出てこない[^1]。え、じゃあSPYDは?VYMは?と検索するも、こちらも出てこない。ここで謎に包まれたまま数ヶ月が経過した。
[^1]:今は検索して出てくる。
いろいろ調べていくうちに、HDVというのは"iシェアーズ コア　米国高配当株 ETF"のことで、SPYDとVYMはそれぞれ"ST SPDRポートフォリオS&P500高配当株式ETF"、"バンガード 米国高配当株式ETF"というらしいということがわかった。

# 米国高配当株ETFで安心!
ほんとにこの銘柄であってるのかな？SPYDなんか名称だとSPDRってなってるじゃんと不安になりながらも、ちょっと値が下がったところを狙って2023年11月にHDV, SPYD, VYMそれぞれ額が均等になるくらいに。
米国株だと税金とかどうなるのかなということで、ちょっとYoutubeとか見たら、米国と日本で二重課税されるとか言ってる。米国にどうやって払うのかわからなわからないけど、なんとかなってるようだし証券会社がいいようにしてくれるんでしょうきっと。将来の自分に託す。がんばれ！自分。

# ちょっと話は戻ります
時間が少し戻って2021年の初夏のこと。
中田敦彦に煽られて全世界株の投資信託(この頃詳しくなくてeMAXISのオルカンじゃなくてニッセイ外国株式インデックスだった)を40万円と金の投資信託(SMT ゴールドインデックス·オープン(為替ヘッジなし))を20万円分買った。当時NISAは"つみたて"か"一般"の2種類しかなくて、インフルエンサーが「"つみたて"を選んでおけ」というので"つみたて"にしていたので、NISAは使わずに買った。
ヘタレな額だが、毎月動向をみて徐々に買い足すつもりだったのだ。結局これっきりとなったが。

時はもどって現在、新NISAには成長投資枠もできたこともあり、今持っている投資信託をNISA口座に付け替えることにした。具体的には、現在保有している投資信託を売っぱらって、同時にNISA口座で買い付けるのだ。
えっと、税金どうなるんだっけ？って口座情報みたら"一般口座"になってる。あれ？特定口座にしてなかったっけ？当時の僕、どうした？

# 米国株買ったときに、為替差益が確定するってみんな知ってた？
当時は米国株購入もNISAに対応していなかったので、こちらも"一般口座"になっていました。じゃあこっちもNISAに載せ替えようかということで、いい加減先送りせずに、改めて確定申告のための準備をしようとなった。
