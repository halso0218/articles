# 一般口座で米国株式を買った顛末

「まあ、細かいことは売るとき考えればいいっしょ」と考えていた私は、こんなに苦労することになるとは思いもしなかった。以下が事の顛末だ。

# 円資産だけじゃヤバイ！
今から4年前の2022年のはじめ頃、いつまでたってもコロナの幻影に怯える民をみて「ちょっともう日本ヤバいんじゃない」という考えに支配されていた私は、資産減損のリスクを減らすためにドルを買うことにした。ちょうど急激な円安が始まった頃だ。
いつものように、思いついてから動き出すまでにひと月ほど時間がかかったが4月6日から毎週5,000円ずつ積み立てることになった。

# ドル預金だけじゃヤバイ！
このころの私は、「現金だけだとインフレに対応できないからヤバイんじゃないか。資産が減損してしまう！」という考えにも支配されており、どんどんたまっていくドルをそのままにしておくことに対して不安になっていた。

とはいえ個別株買うほどの額でもない。ETFでも買っておけばいいや。なんかいい具合に良さそうな株を見繕ってくれてるんでしょ。面倒だし丁度いい、ということで少し調べてみると、高配当株っていうのがよさそうだというのを知った。いわゆるHDV,SPYD,VYMのことだ。

# そんな銘柄ないんだけど！
「さて、ある程度ドルがたまってきたからHDVでも買ってみますか」ということでSBI証券でHDVと検索してみる。ところが検索結果に出てこない※1。え、じゃあSPYDは?VYMは?と検索するも、こちらも出てこない。ここで謎に包まれたまま数ヶ月が経過した。

調べていくうちに、HDVというのは"iシェアーズ コア　米国高配当株 ETF"のことで、SPYDとVYMはそれぞれ"ST SPDRポートフォリオS&P500高配当株式ETF"、"バンガード 米国高配当株式ETF"というらしいということがわかった。

※1:今は出てくる。

# 米国高配当株ETFで安心!
ほんとにこの銘柄であってるのかな？SPYDなんか名称だとSPDRってなってるじゃんと不安になりながらも、ちょっと値が下がったところを狙って2023年11月にHDV, SPYD, VYMそれぞれ額が均等になるくらい購入した。

米国株だと利益にかかる税金とかどうなるのかな？ということで、ちょっとYoutubeとか見たら、米国と日本で二重課税されるとか言ってるが難しくてよくわからない。日本はともかく米国にはどうやって払うの？って思ったけど、証券会社がいいようにしてくれるんでしょうきっと。将来の自分に託す。がんばれ！自分。

# ちょっと話は戻ります
時間が少し戻って2021年の初夏のこと。
中田敦彦に煽られて全世界株の投資信託(この頃詳しくなくてeMAXISのオルカンじゃなくてニッセイ外国株式インデックスだった)を40万円と金の投資信託(SMT ゴールドインデックス·オープン(為替ヘッジなし))を20万円分買った。当時NISAは"つみたて"か"一般"の2種類しかなくて、インフルエンサーが「"つみたて"を選んでおけ」というので"つみたて"にしていたので、NISAは使えずに買った。
ヘタレな額だが、毎月動向をみて徐々に買い足すつもりだったのだ。結局これっきりとなったが。

時はもどって現在、新NISAには成長投資枠もできたこともあり、今持っている投資信託をNISA口座に付け替えることにした。具体的には、現在保有している投資信託を売っぱらって、同時にNISA口座で買い付けるのだ。
えっと、税金どうなるんだっけ？日本の投資信託だから特に何もしなくていいんだよねって口座情報みたら"一般口座"になってる。あれ？特定口座にしてなかったっけ？当時の僕、どうした？

# ついに確定申告の準備に着手！
確定申告確定。
米国高配当株ETFについても当時はもNISAに対応していなかったので、こちらも"一般口座"預りになってた。じゃあついでにこっちもNISAに載せ替えるか～てことで、いい加減先送りせずに、投資信託のとあわせて確定申告の準備をしておくことにした。今年売却予定だから確定申告は来年でいいんだけど、申請時点で売却したときのこと忘れてそうだしね。

いよいよ過去の自分から託された作業を始めよう。証券会社さん期待してますよ！

まずはいくらでドルを買ったかを調べる必要がある。画面で10件ずつ見れるようだが、週次で積立してたから、100件以上データがあるので、これをメモるのは面倒くさい。
当然csvでダウンロードできるっしょって思って探してみるも見当たらない。天下の住信SBIネット銀行、ネット銀行界の雄、どうした？そんな機能もないの？

仕方ないので画面を1画面毎にコピーしてAIに読ませて表形式に変換させることとした。

> 後で気づいたのだが、入出金明細というところから米ドルを指定したら買ったドルについてだけは取得できた。その時の日本円をいくら払ったかはなかったが、積立額を5,000円のまま変更してなかったから、全部5,000円ということで問題ない。

# 米国株買ったときに、為替差益が確定するって知ってた？
さて、次は売却益に対してどういう申告すればいいのかなと調べようと、まずはざっくりCopilotに聞いてみた。すると「ドルで米国株を買うと、そのタイミングでドルを売った扱いとなり為替差益が確定する」と言う。またまたまた～、お得意のハルシネーションだね？税金は売ったときにかかるものなの。だめだぞ？嘘ばっかり言っちゃ。そんなんだとGemini君に乗り換えちゃうよ？

⋯あれ、Geminiどうした？お前もか？　生成AIはこれだからいけない。Webサイトを探して回答をしてくれるPerplexity、お前の出番だ。

⋯ドルでETF買ったときに、為替差益に税金がかかるというので計算することとなった。
さて、為替差益は6万円でした。あと微々たるものだが四半期ごとに配当もある。合計しても20万に全然満たないので確定申告はいらないようだけど、住民税は20万円以下ルールがないらしい。買ったのは2年前だから修正申告か？
うわー面倒くさいー。

# 買った時の履歴を残しておかないと、売った額の95%が利益扱いだよ？
さて今度は米国高配当株ETFの売却益についてだ。ちなみになぜETF買ったときに為替差益が確定するかっていうと、ETFを売却したときには、為替差益のことは考えずにETFの売却益のことだけ考えればよいかららしい。つまり、ETFを売却したドルを円に変える際にたとえ円安になっていて為替差益が出ていたとしても、そこには税金がかからないのだ。

さて買ったときの情報(取引報告書)を入手するかなと思ってみたら、5～7年程度前までのものしかない。⋯いろいろ見た感じ、一般口座の人はそういう履歴は自分で取っておくというのが普通らしい。えー、そういうデータって国税側が把握してくれてるんじゃないの？

もし取得額がわからなくなった場合、売却益(損)ってどうなるの？って調べてみたら、売却額の95%が利益扱いになるらしい！？10年前に110万で買って、それを10万損して100万で売ったとしても、いくらで買ったかわからなければ95万が利益とみなされそこに課税されるのだ。
損するは税金とられるわでふんだり蹴ったり、泣き面に蜂、弱り目に祟り目だ。

そう言えば、親の土地を売ったときもそうだった。バブルも弾けて地価が下がっているから税金払わなくいいはずなのに、買った額かわからなかったせいで２割近い税金でもってかれたなあ。

# さて現状は
さて、これまで見てきていただいた通り、面倒くさがりで動き出すまでに時間がかかる私は、NISAに載せ替えるということで一般口座の投資信託を売却したものの、まだNISAで買えていません。それに米国高配当株ETFに至ってはまだ売ってもいない。

⋯住民税の申告はしないとなあ。
